Binary Rain

Binary Rain is an inspiration of the Matrix Rain by Emily Xie, a creative coder based in the USA. The work was done in P5, a Javascript front-end framework for building quick sketches/visuals. The main objective of the project was to visualize binary symbols trickling from top to bottom. Through this project, I found my love for front-end visuals and I'm definitely going to continue exploring this art.
